# Academic LaTeX Templates

A collection of clean, efficient LaTeX templates for university coursework.

Currently includes a **Standard Assignment Template** optimized for:
* Mathematics & Proofs (Pre-loaded `amsmath`, `amssymb`)
* Computer Science (Syntax highlighting for code blocks)
* Problem Sets (Clear numbering and sectioning)

## Included Templates

### 1. Standard Assignment
A versatile template suitable for weekly homework, problem sets, and short reports.

- **Location:** [`/standard-assignment`](./standard-assignment)
- **Features:**
    - Automatic Header (Name, Course, Date)
    - Clean 'Problem/Solution' environment
    - Python/C++/Java syntax highlighting support

## Quick Start

1.  Clone the repository:
    ```bash
    git clone [https://github.com/YOUR_USERNAME/academic-latex-templates.git](https://github.com/YOUR_USERNAME/academic-latex-templates.git)
    ```
2.  Navigate to `standard-assignment` and open `main.tex` in your LaTeX editor (Overleaf, TeXShop, VS Code).
